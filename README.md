PROJEK KALKULATOR SEDERHANA (JANUAR & GAVRA)

Deskripsi
Project ini merupakan aplikasi kalkulator sederhana berbasis HTML, CSS, dan JavaScript
yang dapat melakukan operasi perhitungan dasar seperti penjumlahan, pengurangan, perkalian, pembagian serta Trigonometri.


Kalkulator kami juga menggunakan fitur tambahan seperti :
theme mode
History
set wallpaper


Tujuan
Melatih penggunaan HTML sebagai struktur halaman.
Mengimplementasikan JavaScript untuk logika perhitungan.
Memahami interaksi antara HTML dan JavaScript.
Mengimplementasikan CSS ke HTML.


Bahasa yang Digunakan
HTML
CSS
JavaScript


Struktur File
kalkulator.html
script.js
style.css


Penjelasan File
index.html
Berisi kerangka utama untuk tampilan yang ada di web kalkulator. Seperti Screen untuk layar, button angka, dan button simbol-simbol aritmatika untuk perhitungan.


style.css
Digunakan untuk mengatur tampilan dan tata letak halaman. Mengatur Warna, posisi, dan beberap hal terkait UI kalkulator.


script.js
Berisi logika perhitungan dan interaksi tombol. Berisi setiap fungsi yang akan digunakan di kalkulator.


Cara Menjalankan
Download atau clone repository
Download extension "Live Server"
Buka folder project
Klik kanan di file index.html
klik "open with live server"


Cara Penggunaan
Klik tombol angka menggunakan keyboard atau touch pad yang ingin di hitung
Pilih operator (+, −, ×, ÷)
Tekan tombol sama dengan (=) atau enter di keyboard 
Hasil perhitungan akan ditampilkan
Klik tombol keyboard (h) untuk menampilkan halaman history, klik lagi untuk menutup halaman history.
klik tombol keyboard (c) untuk menghapus semua history, lalu ada tampilan ada ingin menghapus semua history.
klik tombol keyboard panah kiri atau kanan untuk pindah slide aritmatika biasa dan aritmatika lanjutan.
klik garis tiga di kiri atas untuk mencari fitur lanjutan seperti mengubah wallpaper dan menghapus wallpaper dll.
