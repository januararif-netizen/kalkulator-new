# Projek Calmath Januar dan Gavra

## Deskripsi

Project ini merupakan aplikasi kalkulator sederhana berbasis HTML, CSS, dan JavaScript
yang dapat melakukan operasi perhitungan dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

## Kalkulator kami juga menggunakan fitur tambahan seperti :

1. Theme mode
2. History
3. Setting wallpaper

## Tujuan

1. Melatih penggunaan HTML sebagai struktur halaman.
2. Mengimplementasikan JavaScript untuk logika perhitungan.
3. Memahami interaksi antara HTML dan JavaScript.
4. Mengimplementasikan CSS ke HTML.

## Bahasa yang Digunakan

1. HTML
2. CSS
3. JavaScript

## Struktur File
1. kalkulator.html
2. script.js
3. style.css

## Penjelasan File

1. index.html
Berisi kerangka utama untuk tampilan yang ada di web kalkulator. Seperti Screen untuk layar, button angka, dan button simbol-simbol aritmatika untuk perhitungan.
2. style.css
Digunakan untuk mengatur tampilan dan tata letak halaman. Mengatur Warna, posisi, dan beberap hal terkait UI kalkulator.
3. script.js
Berisi logika perhitungan dan interaksi tombol. Berisi setiap fungsi yang akan digunakan di kalkulator.

## Cara Menjalankan

1. Download atau clone repository
2. Download extension "Live Server"
3. Buka folder project
4. Klik kanan di file index.html
5. klik "open with live server"

## Cara Penggunaan

1. Klik tombol angka menggunakan keyboard atau touch pad yang ingin di hitung
2. Pilih operator (+, −, ×, ÷)
3. Tekan tombol sama dengan (=) atau enter di keyboard 
4. Hasil perhitungan akan ditampilkan
5. Klik tombol keyboard (h) untuk menampilkan halaman history, klik lagi untuk menutup halaman history.
6. klik tombol keyboard (c) untuk menghapus semua history, lalu ada tampilan ada ingin menghapus semua history.
7. klik tombol keyboard panah kiri atau kanan untuk pindah slide aritmatika biasa dan aritmatika lanjutan.
8. Klik garis tiga di kiri atas untuk mencari fitur lanjutan seperti mengubah wallpaper dan menghapus wallpaper dll.
